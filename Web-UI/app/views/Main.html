FAQ

What is ShassAro?

ShassAro is an Open Source Web application that lets you train your cyber security abilities!

Who is it for?

Everyone!
Well, you need to have some basic hacking training and linux skills to help you get started.

Who built it?

ShassAro was build as a graduate project for a Computer Science Bsc in the Collage Of Management of Academic Studies.
And was built by Assaf Aloni, Roi Rav-Hon and Shay Scarlat. More about us here.

How does it work?

Well, that the awesome part.
You and your opponent each get a Linux container (based on Docker).
Each container then customized using Puppet to be unique to you, and only to you.
After that, you start to have fun :-)

Open Source?!

Yep.
And we are glad to accept any contribution.
Either code, tests, bug reports or feature requests.
We are on GitHub.
